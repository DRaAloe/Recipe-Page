# Chocolate Brownie Recipe Website

A simple and visually appealing recipe website built with HTML and CSS. This project showcases a delicious chocolate brownie recipe with a clean layout, custom typography, and a warm bakery-inspired design.

## Features

* Responsive recipe layout
* Custom Google Font (Fredoka)
* Styled ingredients and instructions sections
* Bakery-themed color palette
* High-quality brownie image
* Beginner-friendly HTML and CSS structure

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## About the Project

This project was created as part of my web development learning journey. The goal was to take a basic HTML recipe page and transform it into a more polished and visually appealing website using custom styling and modern design principles.

## How to Run

1. Clone this repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser, or use the Live Server extension.
4. View the recipe website.

## Preview

The website includes:

* A recipe introduction
* An ingredients list
* Step-by-step instructions
* Custom styling for an improved user experience
